Penjelasan:
-----------
- SistemOCRSamboja/:
  Aplikasi web (PHP) untuk upload, review, koreksi, dan manajemen data KTP. File diupload ke public/images, lalu dikirim ke backend Python lewat proses_upload.php.

- PYTHON_OCR/: 
  Backend API (Flask + EasyOCR + OpenCV) menerima file dari PHP, memproses pipeline OCR, dan mengembalikan hasil ke web. Semua proses estraksi ada di core/.

cd C:\laragon\www\TugasAkhirCapstone\SistemOCRSamboja\PYTHON_OCR
python app.py
